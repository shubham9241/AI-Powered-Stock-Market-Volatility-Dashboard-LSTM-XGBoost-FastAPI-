# 🚀 AI-Powered Stock Market Volatility Dashboard

An advanced full-stack financial analytics dashboard that predicts and visualizes stock market volatility in real time using Machine Learning models (LSTM + XGBoost).

## 🔥 Key Features

* 📊 Real-time stock data (NIFTY IT, TCS, INFY, WIPRO)
* 🤖 AI-based volatility prediction (LSTM + XGBoost)
* ⚡ FastAPI backend for high-performance APIs
* 📈 Interactive charts using Chart.js
* 🔄 Live + simulated market data support
* 🚨 Risk detection (Stable / Rising / Critical)

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript, Chart.js
* Backend: FastAPI (Python)
* ML Models: TensorFlow (LSTM), XGBoost

## 💡 Use Cases

* Trading dashboards
* Financial analytics
* AI-based risk monitoring

## ▶️ Run Locally

1. Start backend:

   ```bash
   uvicorn engine:app --reload
   ```
2. Open `index.html` in browser
